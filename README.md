# 👗 Virtual Try-On — AI-Powered Android App

> Real-time body tracking and virtual garment try-on using MediaPipe Pose Detection, built for SPAAK KAI LLP Android Developer Internship Assignment.

**Tech Stack:** Kotlin · MediaPipe · CameraX · Firebase · SQLite

## Features
- 📷 Front & Back camera with real-time pose tracking
- 👕 Garment overlay mapped to body landmarks  
- 📏 AI-based body measurement & size estimation (S–XXL)
- 🎨 Background removal
- ☁️ Cloud sync with Firebase Firestore
- 💾 Local history with SQLite
- 🎯 Live "body locked" tracking status
